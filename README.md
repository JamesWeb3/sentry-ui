# Sentry UI

Agentic workflow blocks and dashboard templates for AI-powered applications.

Copy-paste components built on Next.js, Tailwind v4, and shadcn/ui — styled to the Sentry brand system.

## Templates

| Template | Description |
|---|---|
| [Ecommerce Sales Dashboard](/src/app/templates/ecommerce) | Revenue metrics, charts, products table, streaming AI summary |
| [Content Calendar](/src/app/templates/content) | Monthly planner, status tracking, AI content ideation |

## Stack

- Next.js 15 (App Router)
- Tailwind CSS v4
- shadcn/ui (new-york, neutral)
- Recharts
- Lucide icons
- Motion

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Brand system

Blocks follow the [Sentry brand system](https://github.com/JamesWeb3/sentry-aios/blob/main/skills/sentry-branding/SKILL.md).
